# Node.js & Express — Backend Notes

## Who this is for

This guide is for developers or students who already know basic frontend JavaScript and want to move into backend development with Node.js and Express.

## What I'm learning

- **The Big Picture:** How JavaScript went from browser-only to a powerful server-side tool.
- **Core Concepts:** Asynchronous behavior and why it matters for server performance.
- **Backend Basics:** Setting up a server with Node.js and Express from scratch.
- **API Fundamentals:** Routing, middleware, and creating JSON APIs.

## Prerequisites

- Basic HTML and CSS
- JavaScript variables, functions, and arrays
- No prior backend experience needed

## Why Node.js & Express

- **Unified Language:** Frontend and backend in the same language.
- **High Demand:** Full-stack JavaScript developers are in demand.
- **Efficient Scaling:** Lightweight, fast, and scalable apps.

## What I'll be able to build after this

- Initialize a Node.js project
- Set up a web server from scratch
- Create RESTful API endpoints
- Implement middleware to handle requests
